## FrontEnd

#### Comandos

- `cd frontend` => Para ingresar a la carpeta el front
- `npm install` => Para instalar las dependencias
- `npm start` => Para iniciar el proyecto

#### Diseño

> Colores

```CSS
:root {
    --primaryColor: #0072BC;
    --secondaryColor: #1FA2D6;
    --blackColor: #010101;
    --grayColor: #CCCCCC;
    --whiteColor: #FFFFFF;
}
```

> Estructura de carpetas

```
├───public
└───src
    ├───assets
    │ ├───css
    │ └───img
    └───pages
        └───layouts
```
